🚀 ZenFocus
ZenFocus is a minimalist, high-performance productivity tool designed to minimize distractions and enhance deep-work sessions.

✨ Key Features
Minimalist UI: A distraction-free interface that allows users to focus solely on their current goals.

Modern Aesthetics: Utilizes Glassmorphism and a Midnight-Purple gradient for a premium, soothing user experience.

Smooth Micro-interactions: Custom entry animations provide real-time feedback for task creation and completion.

Responsive Design: Optimized for both mobile and desktop environments.

🛠 Tech Stack
HTML5: For semantic structure.

CSS3: For advanced animations, Glassmorphism, and responsive layout management.

JavaScript (Vanilla): For efficient DOM manipulation and state management.

💡 Why this project?
Modern productivity apps are often overly complex, leading to decision fatigue. ZenFocus follows a "Single-Tasking" philosophy, reducing cognitive load to help users maximize their productivity and maintain focus.

📦 How to Use
Clone this repository.

Open index.html in any modern web browser.

Enter your focus point and press Enter to get started.
